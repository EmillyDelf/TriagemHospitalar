# Triagem Hospitalar

Sistema web de triagem hospitalar desenvolvido com React, TypeScript e Django.

## Objetivo

O projeto tem como objetivo auxiliar profissionais da saúde no processo de triagem de pacientes, permitindo:

- Cadastro de pacientes
- Registro de sinais vitais
- Classificação de risco



# Tecnologias Utilizadas

## Frontend

- React
- TypeScript
- Vite
- CSS
- Font Awesome
